RVC Voice Studio

声音资产管理与相似度分析系统 — 面向心理咨询场景的语音特征提取、
声纹对比与可视化报告生成工具

在线演示 GitHub 仓库

核心功能

人员管理

咨询师与来访者双列管理，支持头像、性别、备注，音频与报告关联展示

音频管理

上传、录音、重命名、删除，支持"原音/变音"标签，自动提取音高、性别、时长等声学指标

声音对比分析

单人分析生成特征报告，双人分析对比咨询师与来访者声音相似度，输出中文判定结论

报告管理

自动生成 HTML / Word 格式报告，支持单人报告与双人对比报告分区管理、预览与导出
